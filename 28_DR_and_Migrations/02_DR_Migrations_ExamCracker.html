The DR & Migration Cheat Sheet

Match the clue to the strategy/service.

ClueAnswer
Cheapest DR, RTO of hours acceptableBackup & Restore
Core DB running/replicated, compute OFFPilot Light
Scaled-down full stack already runningWarm Standby
Near-zero downtime, both regions servingMulti-Site Active-Active
Cross-region relational DB, near-zero RPOAurora Global Database
Migrate DB, minimal downtime, source stays upDMS (full load + CDC)
Migrate to a DIFFERENT DB engineSCT + DMS
Lift-and-shift servers to EC2Application Migration Service (MGN)
Ongoing server replication for DRElastic Disaster Recovery (DRS)
Discover on-prem inventory + dependenciesApplication Discovery Service
Track migration progress centrallyMigration Hub
Move files/PB of data (not databases)DataSync / Snowball / Transfer
