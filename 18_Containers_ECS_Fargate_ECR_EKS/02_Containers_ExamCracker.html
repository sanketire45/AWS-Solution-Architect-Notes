ECS — Cheat Sheet

AWS-native container orchestration.

ConceptWhat
Task DefinitionBlueprint: image, CPU/mem, ports, env, task role
TaskA running instance of a task definition
ServiceKeeps N tasks healthy, registers with ALB
Launch typesFargate (serverless) vs EC2 (you manage nodes)
Task RoleIAM perms for the app in the container (never bake creds)
Task Execution RolePull image from ECR, push logs to CloudWatch
Service Auto ScalingTarget-track CPU/mem/ALB request count
CostECS is free; pay for EC2/Fargate underneath

ECR — Cheat Sheet

Managed container registry.

FeatureDetail
WhatPrivate (& public) managed Docker registry; IAM-secured, encrypted
ScanningBasic, or enhanced via Amazon Inspector (CVEs in OS + deps)
Lifecycle policiesAuto-expire old/untagged images → cost control
ReplicationCross-region & cross-account
Cost$0.10/GB-month + data transfer out

EKS — Cheat Sheet

Managed Kubernetes.

ConceptDetail
WhatManaged Kubernetes control plane (multi-AZ)
WhenOnly if K8s / portability / ecosystem is explicit
NodesEC2 (self/managed node groups) or Fargate (Fargate profiles)
Node scalingCluster Autoscaler or Karpenter
Pod AWS accessIRSA / Pod Identity (not node role)

Kubernetes, managedEKS
