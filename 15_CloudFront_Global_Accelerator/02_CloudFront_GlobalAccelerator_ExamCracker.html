⚔️ CloudFront vs Global Accelerator
CloudFrontGlobal Accelerator

Memory hook
CloudFront caches HTTP. Global Accelerator races TCP/UDP. Both ride AWS edges + get free Shield Standard. GA never caches and never saves transfer cost — it's a performance/availability tool.
